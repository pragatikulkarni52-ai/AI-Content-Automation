# 🤖 AI Multi-Agent Content Automation System

## 📌 Problem Statement
Businesses struggle to create, review, and publish content across multiple platforms while maintaining consistency and compliance. This process is time-consuming and prone to errors.

---

## 💡 Solution
This project presents a simplified *AI multi-agent pipeline* that automates:
- Content Generation
- Compliance Checking
- Content Publishing

Each step is handled by an independent agent, simulating a real-world AI workflow.

---

## ⚙️ Features
- Automated content generation based on topic
- Compliance validation using restricted keywords
- Multi-agent pipeline execution
- Platform selection (Blog, Instagram, LinkedIn)
- Simple and interactive UI using Streamlit

---

## 🧠 System Architecture
The system consists of three agents:

1. *Content Generation Agent*  
   Generates content dynamically based on user input.

2. *Compliance Agent*  
   Checks content for restricted or unsafe keywords.

3. *Publishing Agent*  
   Simulates publishing of approved content.
